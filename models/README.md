# 3D Models Directory

This directory contains external 3D model files used in the game.

## Required Model: Copy Machine

**File needed:** `copier.glb`

### Download Instructions:

1. Visit the Sketchfab model page:
   https://sketchfab.com/3d-models/low-poly-office-copy-machine-f6d56154ad87402e899f8a2df4b114d9

2. Click the "Download 3D Model" button (requires free Sketchfab account)

3. Select the **GLB** format for download

4. Save the downloaded file as `copier.glb` in this directory

Just download as GLB and rename to `copier.glb`.

### Fallback Behavior:

If the model file is not found, the game will display a simple gray box placeholder instead.
